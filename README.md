# 🎰 Pokémon Fortune

> Découvre ton Pokémon du jour — et le message qu'il t'envoie.

🔗 **[pokemon.verglas.cloud](https://pokemon.verglas.cloud/)**

---

## Présentation

Au clic, une roulette de sprites défile puis la carte se retourne pour révéler le Pokémon du jour, son type et un message d'encouragement personnalisé, style fortune cookie.

---

## Fonctionnalités

- Roulette aléatoire parmi 898 Pokémon avec animation de défilement
- Retournement de carte en CSS 3D
- Nom et type traduits en français via double fetch (PokéAPI)
- 898 messages personnalisés stockés en JSON local
- Historique des tirages sauvegardé en localStorage
- Onglets Roulette / Historique
- Design **Gen X Soft Club** — early 2000s web, Windows 98, typo pixel

---

## Stack

`HTML` `CSS` `JavaScript vanilla` `PokéAPI` `JSON` `localStorage` `CSS 3D Transforms`
